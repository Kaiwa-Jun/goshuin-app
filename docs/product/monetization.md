# 収益化

**決定**: フリーミアムモデル（量制限＋機能制限のハイブリッド）

## 無料版 vs プレミアム機能

| 機能 | 無料 | プレミアム |
|---|---|---|
| 御朱印記録 | 月5枚まで | 無制限 |
| 地図表示 | ○ | ○ |
| 画像保存 | 月5枚まで | 無制限 |
| ギャラリー | 直近20件 | 全件 |
| 統計・分析 | × | ○ |
| 県別/地域別の制覇率 | × | ○ |
| データエクスポート(CSV/PDF) | × | ○ |
| 御朱印帳風UIで閲覧 | × | ○ |
| バックアップ・復元 | × | ○ |
| 御朱印帳の複数管理 | 1冊のみ | 無制限 |
| 広告非表示 | × | ○ |

## 価格

- 月額 300円 or 年額 2,400円（2ヶ月分お得）
- 課金ハードルを下げるため300円に設定（缶コーヒー2本分）

## 方針の背景

- 「記録する」というコア体験は無料で制限なく使える（ただし量制限あり）
- 「振り返る」「分析する」「保全する」機能をプレミアムに
- コレクションが増えるほど「ちゃんと管理したい」欲求が生まれる → 課金動機
- 月5枚だと「ちょっと足りない」と感じるライン（月2回参拝で各2-3枚もらうと超える）
- 初めての収益化なので「0→1」を重視
